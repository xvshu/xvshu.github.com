提升用户体验之AB测试（4）——linux升级python到27

此片博客作为工具片存在，帮助大家在安装sixpack时，升级python到2.7

一般linux系统都会预装python，但是一般版本较低，我的测试服务器版本为2.4.4，要升级到2.7.3，大家仅作参考，服务器为redhat版本。

步骤：

1，官网下载地址：http://www.python.org/download/


2，解压

        tar -xzf Python-2.7.3.tgz

3，安装

        将工作目录转到解压好的包下，运行如下命令

        ./configure

        make

        make install

4，测试

  python2.7

    

总结

一般c编写的插件，在linux中，都可以通过以上几步完成安装，大家再遇到类似的问题，都可以用这种方案解决。
