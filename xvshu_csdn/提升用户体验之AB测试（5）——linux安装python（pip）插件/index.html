提升用户体验之AB测试（5）——linux安装python（pip）插件

此片博客作为工具片存在，帮助大家在安装sixpack时，升级python到2.7后安装pip工具，方便安装sixpack

1、pip简介

        在pip的python官网上我们可以看到，pip 是“A tool for installing and managing Python packages.”，也就是说pip是python的软件安装工具，下面是pip的使用方法：


        安装软件：   

                pip install SomePackage


        卸载软件：

                 pip uninstall SomePackage


        更新一个软件：
                pip install --upgrade SomePackage


        查看安装时安装了哪些文件：

                 pip show --files SomePackage


        查看哪些包可以更新：
                 pip show --files SomePackage


        有了pip我们就可以自动的安装 sixpack。


因为安装pip依赖setuptools所以要先安装setuptools


2. 安装setuptools


         (1)，下载
                  wget http://pypi.python.org/packages/source/s/setuptools/setuptools-2.0.tar.gz


         (2)，解压setuptools包
                  tar zxvf setuptools-2.0.tar.gz
                   cd setuptools-2.0


         (3)，编译setuptools
                   python2.7 setup.py build


         (4)，开始执行setuptools安装
                  python2.7 setup.py install

3. 安装 pip

         （1），下载

                 在上面的pip python首页下载pip 包(pip-1.4.1.tar.gz)

         （2），解压

                 tar -xvf pip-1.4.1.tar.gz

         （3），安装软件

                 python2.7 setup.py install 。

         备注：不使用pip安装软件包，也可以用此方法下载、解压后使用 “python setup.py install”安装！


        安装好pip之后，我们就可以直接使用pip安装其他软件包了：

                 sudo pip install sixpack;


总结

        安装python实际和c是一样的，都是固定的那几个命令，大家掌握了之后，就可以自行下载安装，类似的问题，就可迎刃而解，但是pip命令的好处是，帮助我们下载依赖包，省去了我们手动下载，安装的麻烦，大家可自由选择。
