一起学docker（3）安装并运行ubuntu

看了前两篇博客，是否是已经摩拳擦掌了，我们马上开始

1，docker安装（ubuntu系统）

        docker还是对linux做的最好，如果你是windows，建议先安装个虚拟机，装一个linux，然后在进行实验，毕竟线上环境大多是linux。

sudo apt-get install docker.io
静静等待，一路yes下去，docker就在你面前。


2，下载镜像

安装好docker后，启动docker 服务

service docker start


下载ubuntu镜像，我选择网易云docker仓库，因为国外的下载速度比较慢

docker pull hub.c.163.com/library/ubuntu:latest

查看我们有多少镜像：

 docker images

结果如下



3，启动镜像

docker run -ti  hub.c.163.com/library/ubuntu:latest /bin/bash

4，管理容器

查看已经启动的容器

docker ps


进入启动的容器

docker exec -ti 347946a3d77d /bin/bash


总结

        有了这些基础技能，我们就能简单管理一个docker容器了，学习，也就有了乐趣，当然这些也都是前人的经验教训，这里贴出docker学习的网站，供大家选择。
