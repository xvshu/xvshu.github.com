CSS的力量

        CSS（Cascading Style Sheet）级联样式表，是一种美观网页设计的解决方案，也是W3C推荐的标准，他可以是我们的网页设计更灵活，更美观，使设计人员对内容的设计和样式的设计分离，使设计逻辑简单，更有利于网页的开发，掌握CSS是当下网页前台设计人员必不可少的一个技能，在此我们共同研究和总结关于CSS的学习，让我们在成为大虾的道路上再迈出坚实的一步！


 一，CSS简介


        百度百科：

        级联样式表（Cascading Style Sheet）简称“CSS”，通常又称为“风格样式表（Style Sheet）”，它是用来进行网页风格设计的。比如，如果想让链接字未点击时是蓝色的，当鼠标移上去后字变成红色的且有下划线，这就是一种风格。通过设立样式表，可以统一地控制HTML中各标志的显示属性。级联样式表可以使人更能有效地控制网页外观。使用级联样式表，可以扩充精确指定网页元素位置，外观以及创建特殊效果的能力。

        维基百科：

        层叠样式表（英语：Cascading Style Sheets，简写CSS），又称串样式列表，一种用来为结构化文档（如HTML文档或XML应用）添加样式（字体、间距和颜色等）的计算机语言，由W3C定义和维护。目前最新版本是CSS2.1，为W3C的推荐标准。CSS3现在已被大部分现代浏览器支持，而下一版的CSS4仍在开发过程中。

        

        总结：

        CSS是将样式的设计同内容的设计分离，统一管理样式的设计，方便编程人员的管理的工具！


二，CSS学习攻略

        对CSS的学习要掌握这么几个原则：

        1，实用为原则，所有接触到的知识，自己动手去简单实现

        2，对比原则，任何知识都有其领域，在同一个领域内要多对比相似的两个知识点，CSS的学习主要对比HTML，XML等标记语言！

        3，及时原则，初级学习完后，接触新的知识，要及时归队，形成总结，打造自己的知识库，以备不时之需啊！
